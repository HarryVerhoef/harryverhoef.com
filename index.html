C Subset Compiler

This project was part of a Compiler Design module that I took in the final-year at Warwick, and it is a LL(2) recursive-descent compiler that translates a language which uses a subset of the C syntax into three-address code. The compiler first performs a lexical analysis of the source code, then parses the resultant tokens into an Abstract Syntax Tree (AST), and finally generates the three-address code.
